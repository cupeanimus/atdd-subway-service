### 요구 사항
- [X] LineSectionAcceptanceTest 리팩터링
- [X] LineService 리팩터링

- [X] 한번에 많은 부분을 고치려 하지 말고 나눠서 부분부분 리팩터링하기
- [X] 전체 기능은 인수 테스트로 보호한 뒤 세부 기능을 TDD로 리팩터링하기
